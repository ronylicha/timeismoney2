Fonctionnalités complètes

Tout ce dont vous avez besoin pour gérer vos projets, votre temps et vos factures

Suivi du temps

Timer en temps réel, feuilles de temps hebdomadaires, rapports détaillés et exportation des données.

Gestion de projets

Projets, tâches, Kanban, assignations d'équipe et suivi de l'avancement en temps réel.

Facturation

Création de factures, devis, envoi par email, génération PDF et intégration Chorus Pro.

Gestion des clients

Base de données clients, contacts, historique et analyse des relations commerciales.

Rapports & Analytics

Tableaux de bord dynamiques, graphiques interactifs, statistiques et exportations FEC.

Sécurité

2FA, permissions granulaires, audit logs, chiffrement et conformité RGPD.

Technologies modernes

Construit avec les dernières technologies pour performance et maintenabilité

Backend

  • Laravel 12 - Framework PHP moderne
  • PHP 8.3+ - Performance et typage
  • MySQL 8.0 - Base de données relationnelle
  • Sanctum - Authentication API
  • Queues & Jobs - Tâches asynchrones

Frontend

  • React 18 - Interface utilisateur réactive
  • TypeScript - Typage statique
  • Tailwind CSS v4 - Design moderne
  • TanStack Query - Gestion d'état serveur
  • Vite 7 - Build tool ultra-rapide
